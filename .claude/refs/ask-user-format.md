# ASK_USER_FORMAT
> 모든 Skill/Agent의 승인 요청, 선택지 제시 표준 형식.

## 구조 (순서 고정)
```
1. CONTEXT:        지금 어느 단계인가 (1줄)
2. ISSUE:          무엇을 결정해야 하는가 (전문용어 없이)
3. RECOMMENDATION: 권장 선택지 + 이유 (1줄)
4. OPTIONS:        선택지 (알파벳, 1줄씩)
```

## 형식
```
📌 [단계명] | [작업명]
상황: (지금 무슨 일인지 한 문장)
결정: (무엇을 선택해야 하는지)
권장: [X] — (이유 한 줄)
A) (선택지)
B) (선택지)
C) (선택지, 있으면)
```

## 규칙
```
- 이슈 하나당 질문 하나 (배치 금지)
- SR HIGH 이슈는 반드시 개별 질문
- 권장 선택지는 항상 명시
- 기술 용어는 쉬운 말로 풀어서
```
